# Train Schedule

1. Write the name of the train you want.
2. Write the destination.
3. Write the first arrival time in HH:MM format.
4. Write the frequency of the train.

After clicking submit button, following things will get displayed.

1. Train Name
2. Destination
3. Frequency of the Train
4. When the next train will arrive.
5. How many minutes you have to wait til the next train comes.

All of the entered data will be saved in the database.
